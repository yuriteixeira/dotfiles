## Creating plans

- Be concise: AI needs guidance, but isn't dumb, so no need to be verbose unecessarily.
- Focus your energy on digging into the what's unique to the problem space.
- If something is unclear, ask the most important questions to the user to refine the plan.
- Write the plan in ./docs/plans and if there's a numeric prefix for the existing ones (001-plan-a.md), continue to use it (eg: 002-plan-being-created.md)

## Executing plans

- If something is unclear, ask the most important questions to the user to refine the plan, let the user review it and when they are done, execute it. 

## Misc 

- Never forget to respect other guidelines in AGENTS.md and skills while creating or executing plans.
